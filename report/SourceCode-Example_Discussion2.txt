LIGUAGEM GENERAL PURPOSE (POTATOES)

	Após as considerações do ficheiro anterior, colocou-se em questão a utilização
do simbolo [...] para demarcar as unidades em operações com literais. Sendo o
problema a confusão criada entre essa simbologia e o normal uso do mesmo simbolo
em arrays. Ao debater este assunto foi considerada a substitução por plicas como
a possibilidade de eliminar de todo a funcionalidade visual e prática desse elemento.
Assim, abaixo, apresenta-mos vários exemplos de código fonte lado a lado para
comparação daquele que melhor veicula a informação pretendida.

Consideremos a seguinte declaração de unidades:

constants {
	KILO 	[k]	: 10^3
	MILI 	[m]	: 10^(-3)
	PI		[pi]: 3.14
}
units {
	meter "m"
	inch "inch"
	yard "yd"
	distance "d": (2)meter | (0.025)inch | (0.9)yard
	mile "mil" : (0.75)distance | (2)meter
	
	animais "a" : (1)pessoa | (1)vaca
	legs	"l" : (2)pessoa | (4)vaca
	
	velocity "in/s" : inch /second
	velocity "m/s" : (0.0125)distance / second
	
	
	
	time		[s]
	velocity		: distance / time
}
